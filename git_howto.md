# Подсказка по GIT

Команда смены дирректории
```sh
cd c:\folder__name
```
Команда отображения текущей директории MacOs, Linux
```sh
pwd
```
Листинг текущей директории Windows:
```sh
dir
```
MacOs, Linux
```sh
ls
```
Создание репозитория
```sh
git init
```
Статус папки
```sh
git status
```
Команда для добавления и сохранения файла 
```sh
git add \file name
```
